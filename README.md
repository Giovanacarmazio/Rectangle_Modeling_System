# Rectangle Modeling System

Este repositório contém o projeto **Rectangle Modeling System**, que implementa a modelagem de retângulos em Python. O sistema calcula propriedades de um retângulo, como área e perímetro, com base nas dimensões fornecidas pelo usuário.

## Descrição do Projeto

O objetivo do **Rectangle Modeling System** é oferecer uma maneira simples de calcular as propriedades geométricas de um retângulo, utilizando os valores de comprimento e largura fornecidos pelo usuário.

## Funcionalidades

- Cálculo da área do retângulo.
- Cálculo do perímetro do retângulo.

## Tecnologias Utilizadas

- **Python**

## Estrutura do Repositório

- `rectangle_model.py`: Código principal que executa os cálculos de área e perímetro.
- `docs/`: Documentação explicativa sobre os cálculos realizados.

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Giovanacarmazio/Modelacao-Retangulo.git
   ```

2. Execute o script:
   ```bash
   python rectangle_model.py
   ```

3. Insira as dimensões do retângulo (comprimento e largura) para obter os cálculos de área e perímetro.
